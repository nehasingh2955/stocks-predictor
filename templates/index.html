Stocks Predictor

Welcome {{ username }}

This is a tool to predict future stock prices. Users can add and remove companies they want predictions for from their "portfolio". Price is predicted by performing sentiment analysis using a Naive Bayes Classifier and the natural language toolkit along with a non-linear regression model on a stock's one month price chart. Using News API to get relevant news and Stock News API to get training data.

Disclaimer

This is meant to be a fun project and users should not base their decisions off these predictions.
