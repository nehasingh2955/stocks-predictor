Stocks Predictor

A tool to predict future stock prices for various tech companies i.e. Google, Tesla, Amazon, Uber etc. Price is predicted by performing sentiment analysis using a Naive Bayes Classifier and the natural language toolkit along with a non-linear regression model on a stock's one month price chart. Using News API to get relevant news and Stock News API to get training data. The source code can be found here.
